# Interactive Navigation Menu

An interactive navigation bar that changes its style on scroll and on hover, built using pure HTML, CSS, and JavaScript.

## Features
- Fixed navigation bar
- Background and shadow change on scroll
- Smooth hover color effects on menu items
- Clean and simple UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
- JavaScript listens to the window scroll event
- A CSS class is added or removed based on scroll position
- CSS handles hover effects and transitions
